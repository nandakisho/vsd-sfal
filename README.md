# vsd-sfal
Day 1: Install yosys, iverilog and gtkwave
1.1: install yosys

$ git clone https://github.com/YosysHQ/yosys.git
$ cd yosys
$ brew install make (If make is not installed please install it) 
$ $ brew tap Homebrew/bundle && brew bundle
$ make config-gcc
$ make 
$ brew make install
